# Tính lũy thừa

Cho số nguyên $a$ và $n$.

***Yêu cầu:*** Bạn hãy viết chương trình **đệ quy** tính giá trị $a^n$.

## Input

- Dòng duy nhất chứa hai số nguyên $a$ và $n$.

## Constraints

- $1 \le a \le 100$
- $1 \le n \le 100$

## Output

- Giá trị $a^n$, vì kết quả có thể rất lớn nên bạn chỉ cần in ra kết quả modulo $10^9 + 7$.

## Sample input 1

```
2 5
```

## Sample output 1

```
32
```

## Sample input 2

```
100 100
```

## Sample output 2

```
424090053
```
