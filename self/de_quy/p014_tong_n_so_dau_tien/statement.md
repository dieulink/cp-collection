# Tổng n số đầu tiên

Cho số nguyên $n$.

***Yêu cầu:*** Bạn hãy viết chương trình đệ quy tính tổng $1 + 2 + \ldots + n$.

## Input

- Chứa duy nhất số nguyên $n$.

## Constraints

- $1 \le n \le 100$

## Output

- Tổng $1 + 2 + \dots + n$

## Sample input 1

```
4
```

## Sample output 1

```
10
```
