# Ước chung lớn nhất

Cho hai số nguyên dương $a$ và $b$. 

***Yêu cầu:*** Hãy viết chương trình **đệ quy** tính ước chung lớn nhất của $a$ và $b$.

## Input

- Dòng đầu tiên chứa số nguyên $a$.
- Dòng thứ hai chứa số nguyên $b$.

## Constraints

- $1 \le a, b \le 10^9$

## Output

- Ước chung lớn nhất của $a$ và $b$.


## Sample input 1

```
3 4
```

## Sample output 1

```
1
```

## Sample input 2

```
16 12
```

## Sample output 2

```
4
```
